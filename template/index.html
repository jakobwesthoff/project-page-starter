Documentation

Options

Option Description Default
-o, --output <path> Specify output directory Current directory
-v, --verbose Enable verbose logging Off
-c, --config <file> Use custom config file ~/.config/project/config.toml
-q, --quiet Suppress all output Off
-h, --help Print help information -

Examples

# Process a single file
project-name document.txt

# Custom output location
project-name document.txt --output ~/Documents/results

# Verbose mode for debugging
project-name document.txt --verbose

# Using a config file
project-name document.txt --config my-config.toml

Configuration

Create a configuration file at ~/.config/project/config.toml: 

# config.toml
output_dir = "~/Documents/output"
verbose = false
max_threads = 4
